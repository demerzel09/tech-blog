# chatgpt-share-summary-zenn

## 概要
このディレクトリは、ChatGPT共有内容をもとにしたZenn記事作成用の作業フォルダです。

- 元資料（PDFなど）は `docs/` に配置
- 本文ドラフトやメモはこのフォルダ配下で管理
- VS Codeで開いて、そのまま執筆・更新できる構成
